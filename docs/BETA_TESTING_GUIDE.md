### Before Reporting

2. Try to reproduce the issue

### Creating a Good Bug Report

**Title**: Clear, descriptive summary
```
Bad:  "App doesn't work"
Good: "Browser module crashes when opening PDF files"
```

**Include**:
- Steps to reproduce
- Expected behavior
- Actual behavior
- Screenshots/recordings (if applicable)
- System information
